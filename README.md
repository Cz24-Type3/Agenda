Um aplicativo mobile que eu criei para testar as minhas habilidades.

Nele você pode salvar, deletar e editar "Cards" de contatos.

    Como usar?
1- Instalas as dependências
    npm install
2- Rodas o Expo
    npm start
3- Rodar em Web ou em emulador
    Pressionar W para rodar em web
    Pressionar A para rodar no emulador

Buildar APK
    eas build
